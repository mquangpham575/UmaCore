# Privacy Policy

**Last updated: April 1, 2026**

This Privacy Policy describes what data UmaCore collects, how it is used, and how it is stored.

## 1. Data We Collect

UmaCore collects and stores only the minimum data required to function:

| Data | Purpose |
|------|---------|
| Discord Server ID | Identify which server a configuration belongs to |
| Discord Channel IDs | Know where to send quota reports and notifications |
| Discord User IDs | Link Discord accounts to club members for DM notifications |
| Club member names and fan counts | Core quota tracking functionality |
| Server configuration settings | Store per-server preferences (quotas, schedules, thresholds) |

We do **not** collect message content, passwords, payment information, or any other personal data.

## 2. How Data Is Used

Collected data is used exclusively to provide UmaCore's features:

- Tracking daily fan-earning quotas per club
- Sending bomb warnings and quota reports to configured channels
- Delivering personal DM notifications to linked users
- Generating progress charts

Data is never sold, shared with third parties, or used for advertising.

## 3. Data Storage

Data is stored in a PostgreSQL database. Access is restricted to the bot and its operators. We take reasonable precautions to protect stored data, but cannot guarantee absolute security.

## 4. Data Retention

Data is retained for as long as UmaCore is active in your server. If UmaCore is removed from a server, you may request deletion of your server's data by contacting us (see Section 6).

## 5. Your Rights

You may request:

- A copy of the data stored for your server or Discord account
- Deletion of your data

To make a request, contact us using the information in Section 6.

## 6. Contact

For privacy-related requests or questions, open an issue on [GitHub](https://github.com/oHaruki/UmaCore) or contact the developer via Discord.

## 7. Changes to This Policy

We may update this Privacy Policy from time to time. Continued use of UmaCore after changes are posted constitutes acceptance of the updated policy.
